Project Summary

Welcome! This site shows some initial data exploration visuals and accompanying analysis centering around the question: are cities near the equator actually hotter? What other observable weather patterns might there between cities when considering their latitude?

Site Directory:

To get an initial understanding of the data itself, use the "Data" tab in the upper right corner to see the full table of cities. To see the weather data that was generated, below are four individual links to the specific visuals and a general analysis of each; these can also be found in the dropdown "Plots" menu in the navigation pane. For more information on how the data itself was generated and what it means, head over to the "Comparisons" page using the navigation bar.
